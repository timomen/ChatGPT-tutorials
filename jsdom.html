JavaScript DOM

De Document Object Model (DOM) is een programmeerinterface voor webpagina's. Het biedt een hiërarchische representatie van de HTML-structuur van een webpagina, waarmee JavaScript kan worden gebruikt om dynamisch de inhoud en het uiterlijk van een webpagina te wijzigen. Hieronder volgt een tutorial over het gebruik van JavaScript en de DOM.

1. Het selecteren van HTML-elementen

JavaScript kan worden gebruikt om HTML-elementen te selecteren en te manipuleren. Dit kan worden gedaan met behulp van verschillende methoden, waaronder:

getElementById

De getElementById methode retourneert het HTML-element met de opgegeven id-attribuutwaarde.

const element = document.getElementById("example");

getElementsByClassName

De getElementsByClassName methode retourneert een array-achtig object met alle HTML-elementen die de opgegeven klasse hebben.

const elements = document.getElementsByClassName("example");

getElementsByTagName

De getElementsByTagName methode retourneert een array-achtig object met alle HTML-elementen van het opgegeven tagtype.

const elements = document.getElementsByTagName("p");

querySelector

De querySelector methode retourneert het eerste HTML-element dat overeenkomt met de opgegeven selector.

const element = document.querySelector(".example");

querySelectorAll

De querySelectorAll methode retourneert een NodeList-object met alle HTML-elementen die overeenkomen met de opgegeven selector.

const elements = document.querySelectorAll("p.example");

2. Het wijzigen van HTML-elementen

Nadat een HTML-element is geselecteerd, kan het worden gewijzigd met behulp van verschillende eigenschappen en methoden.

innerHTML

De innerHTML eigenschap kan worden gebruikt om de inhoud van een HTML-element te wijzigen. Het kan worden ingesteld op een nieuwe HTML-string, inclusief HTML-tags.

const element = document.getElementById("example"); element.innerHTML = "<p>Dit is een nieuwe paragraaf.</p>";

textContent

De textContent eigenschap kan worden gebruikt om de tekstinhoud van een HTML-element te wijzigen.

const element = document.getElementById("example"); element.textContent = "Dit is nieuwe tekst.";

style

De style eigenschap kan worden gebruikt om de CSS-stijl van een HTML-element te wijzigen.

const element = document.getElementById("example"); element.style.backgroundColor = "red";

classList

De classList eigenschap kan worden gebruikt om de klasse van een HTML-element te wijzigen.

const element = document.getElementById("example"); element.classList.add("new-class");

3. Het toevoegen van HTML-elementen

Nieuwe HTML-elementen kunnen worden toegevoegd aan een webpagina met behulp van de createElement en appendChild methoden.

const element = document.createElement("p"); element.textContent = "Dit is een nieuwe paragraaf."; const container = document.getElementById("container"); container.appendChild(element);

In dit voorbeeld wordt een nieuwe paragraaf gemaakt met behulp van de createElement methode en vervolgens toegevoegd aan het HTML-element met het id-attribuut "container" met behulp van de appendChild methode.

4. Het verwijderen van HTML-elementen

HTML-elementen kunnen worden verwijderd van een webpagina met behulp van de removeChild methode.

const element = document.getElementById("example"); const container = document.getElementById("container"); container.removeChild(element);

In dit voorbeeld wordt het HTML-element met het id-attribuut "example" verwijderd uit het HTML-element met het id-attribuut "container" met behulp van de removeChild methode.

5. Het toevoegen van event listeners

JavaScript kan worden gebruikt om event listeners toe te voegen aan HTML-elementen. Dit maakt het mogelijk om JavaScript-code uit te voeren wanneer bepaalde gebeurtenissen optreden, zoals een muisklik of toetsaanslag.

const button = document.getElementById("button"); button.addEventListener("click", function() { alert("Hallo wereld!"); });

In dit voorbeeld wordt een event listener toegevoegd aan het HTML-element met het id-attribuut "button". Wanneer dit element wordt geklikt, wordt de anonieme functie uitgevoerd, die een berichtvenster met de tekst "Hallo wereld!" weergeeft.

6. Het wijzigen van attributen

HTML-attributen kunnen worden gewijzigd met behulp van de setAttribute methode.

const element = document.getElementById("example"); element.setAttribute("class", "new-class");

In dit voorbeeld wordt het class-attribuut van het HTML-element met het id-attribuut "example" gewijzigd in "new-class".

tagName

De tagName eigenschap retourneert de tag van een HTML-element.

const element = document.getElementById("example"); const tag = element.tagName; console.log(tag);

In dit voorbeeld wordt de tag van het HTML-element met het id-attribuut "example" verkregen met behulp van de tagName eigenschap.

id

De id eigenschap retourneert de waarde van het id-attribuut van een HTML-element.

const element = document.getElementById("example"); const id = element.id; console.log(id);

In dit voorbeeld wordt de waarde van het id-attribuut van het HTML-element met het id-attribuut "example" verkregen met behulp van de id eigenschap.

classList

De classList eigenschap retourneert een lijst van de klassen van een HTML-element.

const element = document.getElementById("example"); const classes = element.classList; console.log(classes);

In dit voorbeeld wordt een lijst van de klassen van het HTML-element met het id-attribuut "example" verkregen met behulp van de classList eigenschap.
